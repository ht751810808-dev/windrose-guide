# tomodachi-life.org

Fan site for Tomodachi Life — covering the original 3DS game and Tomodachi Life: Living the Dream (Switch).

**Domain:** [tomodachi-life.org](https://www.tomodachi-life.org)

## Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Inline styles + CSS custom properties
- **Fonts:** Fredoka + Nunito (Google Fonts)
- **Deployment:** Vercel
